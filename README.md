## **Document abstract**

Real-estate platforms user-generated data (UGD) offer valuable insights into housing search behavior, but they are heavily affected by noise generated by recreational or low-engagement online searchers. This noise complicates the identification of serious searchers and limits the reliability of UGD for market analysis. This study investigates whether the individual distributions of listings viewed by searchers contain behavioral signals that can help distinguish serious searchers from recreational ones, beyond traditional indicators such as account creation or seller contact. We combine a set of variability and revisit metrics describing searchers' viewing patterns, and we find that revisits emerge as the strongest behavioral signal of searcher engagement and the primary driver of seller contact, whereas search variability plays a secondary but complementary role. The behavioral structure reduces to two dominant dimensions, revisit intensity and timing and search variability, and these two dimensions are good proxies for contact initiation. Our results demonstrate that UGD can be refined to mitigate recreational-search noise and improve the identification of serious search behavior, and that this purely behavior-based approach makes it possible to analyse UGD even in the absence of additional information such as contact initiation or account creation.

**Keywords** : Housing search, real estate platforms, user-generated data, housing market, Web data

## Organization {#organization}

- `in`: folder containing all the input data files.
  - `carto`folder containing the input shapefiles.
  - `extern_data`: folder containing the external datasets for variables corrections.
  - `raw`: folder containing the raw datasets (listings' features and events' datasets).
- `out`: folder containing the outputs of the processing.
  - `pdf`: pdf files.
  - `Rdata`: Rdata files.
  - `svg`: svg files.
  - `tex`: tex files (output in Latex format).
- `src`: folder containing the source code.
  - `plot`: `R` scripts for plots.
    - `map_plot.R`: `R` script to plot Online search intensity by city.
    - `sample_search_cluster_plot.R`: `R` script to plot serious search graphics.
    - `serious_search_pdf_plot.R`: `R`script to save graphics in pdf format.
    - `serious_searcher_svg_plots.R`: `R`script to save graphics in svg format.
  - `processing`: `R` scripts for data processing ad hoc functions.
    * `cartography`: folder for cartography processing
      * `carto.R`: functions used for cartographic data processing.
    * `data_cleaning`: folder containing cleaning data scripts.
      * `import_SL_event_data.R`: `R`script to import events yearly data.
      * `listing_features_data_SL.R`: `R`script to process SeLoger dataset.
  - `serious_search`: `R` scripts for the analysis of serious search.
